# 🎬 CineStream - Ver Películas y Series Online

Plataforma de streaming que usa la API de TMDB para mostrar información de películas y series, con reproductor de video integrado.

## ✨ Características

- 🎥 Catálogo completo de películas y series
- 🔍 Búsqueda en tiempo real
- 🎬 Reproductor con múltiples servidores
- 📱 Diseño responsive (móvil y escritorio)
- 🎨 Interfaz moderna tipo Netflix
- 🌐 Filtros por género y ordenamiento
- 📺 Selector de temporadas y episodios para series

## 🚀 Cómo Subir a GitHub Pages

### Paso 1: Crear cuenta en TMDB
1. Ve a https://www.themoviedb.org/signup
2. Crea una cuenta gratuita
3. Ve a tu perfil → Configuración → API
4. Solicita una API Key (es gratis)
5. Copia tu **API Key (v3 auth)**

### Paso 2: Crear repositorio en GitHub
1. Ve a https://github.com/new
2. Nombre del repositorio: `cine-stream` (o el que quieras)
3. Marca como **Public**
4. Haz clic en "Create repository"
